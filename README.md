# Ducrong UI

## 快速上手

### 前提条件
> **注意**：
> 需要 [Node.js](https://nodejs.org/en/) >= 8.6

### 安装依赖

```bash
npm install # yarn install
```

### 本地启动

```bash
npm run docs:dev # yarn docs:dev
```

### 生产打包

```bash
npm run docs:build # yarn docs:build
```

VuePress 会在 [http://localhost:1000](http://localhost:1000) 启动一个热重载的开发服务器。

> **注意**：
> 新增/删除 .md 文件以及 vuepress 配置变化需要重新启动服务器

了解项目 [目录结构](directory.html) 和 VuePress 中的 [基本配置](https://vuepress.vuejs.org/zh/reference/config.html)。
