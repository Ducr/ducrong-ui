# 目录结构

文档 docs 基于 [vuepress](https://vuepress.vuejs.org/zh/guide/introduction.html)

```JSON
├── dist                    // 文档构建代码
├── docs
│   ├── .vuepress
│   ├── README.md
│   ├── components          // 组件文档
│   .
│   .
│   └── README.md
├── packages                // 组件源码
│   .
│   .
└── package.json
```
